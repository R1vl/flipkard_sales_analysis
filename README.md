# Анализ продаж товаров Flipkard

**Цель проекта:** выявить ключевые закономерности в продажах товаров Flipkard и дать рекомендации для бизнеса.  
**Автор:** Привалова Анастасия  
**Инструменты:** Python (pandas, matplotlib, seaborn, statsmodels), SQL, Jupyter Notebook  

---

## О проекте

Проект представляет собой полный цикл аналитики данных:  
1. Проверка корректности данных и очистка  
2. Exploratory Data Analysis (EDA)  
3. Углублённый анализ и создание бизнес-признаков (feature engineering)  
4. Моделирование и проверка гипотез  
5. Симуляция A/B теста  
6. SQL-аналитика  

Проект выполнен на учебных данных и демонстрирует навыки анализа, визуализации и интерпретации результатов для бизнес-задач.

---

## 1. Проверка и очистка данных
- Всего товаров: 80 000, 25 колонок  
- Пропущенные значения: только `size` (заполнены `Unknown`)  
- Дубликатов нет, типы данных корректны  
- Уникальность идентификаторов подтверждена  

**Вывод:** данные готовы для анализа, отсутствуют критические ошибки.

---

## 2. Exploratory Data Analysis (EDA)
- **Цены и скидки:** широкий диапазон цен, без выбросов; скидки дискретные (0–50%), медиана ≈ 20%  
- **Рейтинг товаров:** сосредоточен вокруг 3, без аномалий  
- **Категории:** топ-3 категории формируют значимую долю выручки, но доминирование не критично  
- **Взаимосвязи:** цена, скидка, рейтинг не оказывают линейного влияния на выручку  

**Вывод:** структура спроса сложная, вероятно влияние скрытых факторов (сезонность, маркетинговые акции, ассортиментная стратегия).

---

## 3. Углубленный анализ (Feature Engineering)
- Созданы новые признаки:
  - Ценовой сегмент (low/mid/high)  
  - Временные признаки (месяц, год размещения)  
  - Метрики скидок и эффективности продаж (discount_amount, discount_ratio, sell_through_rate)  
  - Логистические признаки (скорость доставки)  
- **Выручка по ценовым сегментам:** медианная выручка растёт с изменением ценового сегмента low - mid - high 
- **Рейтинг:** не оказывает значимого влияния на продажи  
- **Сезонность:** пики продаж в январе, марте, мае, июле, августе, ноябре  
- **Скорость доставки:** минимальное влияние на выручку  

**Вывод:** ключевые факторы — ценовой сегмент и сезонность.

---

## 4. Моделирование
- Множественная линейная регрессия (OLS) для выручки:
  - Ценовой сегмент — значимый фактор  
  - Рейтинг, скорость доставки, месяц — незначимые  
  - R² ≈ 0.35 (интерпретируемая модель, не для точного прогноза)  

**Вывод:** выручка определяется главным образом ценовым сегментом. Рекомендация — фокус на High-сегменте для увеличения дохода. Гипотеза о сезонности не подтвердилась.

---

## 5. Симуляция A/B теста (изменение цены ±5%)
- Сегмент: mid-price, высокий рейтинг, возвратный товар, надежный продавец  
- Результаты:
  - Разница выручки ≈ 3 млн  
  - p-value = 0.233 – статистически значимого эффекта нет  

**Вывод:** небольшие корректировки цены безопасны, влияние на суммарную выручку минимальное.

---

## 6. SQL-аналитика
- Всего товаров: 80 000  
- Топ-5 категорий по количеству товаров: Toys, Beauty, Fashion, Electronics, Sports  
- Средняя цена и рейтинг брендов: цены ≈30 000, рейтинг ≈3.0, рынок конкурентный  
- Топ-10 товаров по выручке: Nike Series 426 — лидер  
- Выручка по ценовым сегментам: high-price сегмент приносит наибольшую выручку  

---

## 7. Итог
- Основной драйвер выручки — ценовой сегмент товара  
- Визуально наблюдаемая сезонность не подтверждается статистически.
Сезонные колебания объясняются структурой данных (количество объектов, длительность месяца), а не реальными изменениями средней выручки.
- Рейтинг и скорость доставки не оказывают влияние на выручку  
- Малые изменения цен (+/-5%) не критично влияют на доход  
- Рекомендации для бизнеса: приоритетные направления для оптимизации дохода — корректировка ассортимента и ценовой политики
